# AI Toolkit CLI

AI Toolkit CLI built with Python for working with AI tools.

---

## Repository Structure

```
AI-Toolkit-CLI/
├── main.py
├── README.md
├── requirements.txt
├── .env.example
```

---

## Prerequisites

| Tool     | Version |
|----------|---------|
| Python   | ≥ 3.9   |
| pip      | Latest  |

---

## Quick Start

```bash
# 1. Clone the Repository 
Git clone https://github.com/Renz-Victa/AI-Toolkit-CLI
cd AI-Toolkit-CLI

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the CLI
python main.py
```

---

### Configuration

Create a .env file in the project root, if the toolkit needs an API key

```
API_KEY=your_api_key_here
```

After that, load it in main.py file before you run the CLI.
